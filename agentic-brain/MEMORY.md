# Memory

## Decisions

- The project started as a simple knowledge bot MVP.
- Documentation was created before implementing retrieval logic.
- A lightweight keyword-based approach was selected for the first version.

## Lessons Learned

- Keeping documentation organized simplifies future development.
- Agent context files help future contributors understand the project faster.

## Future Improvements

- Add semantic search using embeddings.
- Add conversation history.
- Add API support.
- Add web interface.
