# Project Brief

## Project Name
Mini Company Knowledge Bot

## Goal
Build a simple knowledge bot that answers user questions based on internal company documents stored in the docs directory.

## MVP Scope

The system should:

- Load company documents from the docs folder
- Accept a user question
- Search the available documents
- Return an answer based on document content

## Included Documents

- FAQ
- Leave Policy
- Product Information

## Future Improvements

- Semantic Search
- Embedding-based Retrieval
- Web Interface
- API Support
- Conversation Memory

## Success Criteria

A user can ask questions about company policies or products and receive relevant answers from the available documentation.
