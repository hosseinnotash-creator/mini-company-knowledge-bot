# Mini Company Knowledge Bot

A simple knowledge bot that answers questions using company documents.

## Features

- Document ingestion
- Question answering
- Agent-friendly project structure

## Project Structure

docs/
agentic-brain/
app.py

## Run

```bash
python app.py
```

## Example

Question:

How many annual leave days do employees receive?

Answer:

Employees receive 20 annual leave days.
```
